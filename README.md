⏬ Install
```shell
npm i vue-sparkly-text
```

# 🪡 Usage

```javascript
import { ReactSparklyText } from "react-sparkly-text"
``` 

```html
<p>Lorem ipsum dolor <ReactSparklyText>sparkly text</ReactSparklyText> sit amet.</p>
```
